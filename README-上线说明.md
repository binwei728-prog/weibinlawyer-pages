## 上线前需要替换

1. 将 `index.html`、文章页、`robots.txt`、`sitemap.xml` 和 `llms.txt` 中的 `https://example.com/` 替换为正式域名。
2. 将结构化数据中的 `sameAs` 占位链接替换为真实公开主页：
   - 公众号主页或代表文章链接；
   - 知乎主页；
   - 小红书主页；
   - 视频号或其他公开主页；
   - 律所官网个人页。
3. 如后续有正式头像或执业照，可增加到首页，但应保持专业、清晰、真实。
4. 联系方式以上线前确认的公开联系方式为准。

## AI可见度要点

网站已经内置：

- 页面标题；
- 页面描述；
- 关键词；
- 作者信息；
- Person 结构化数据；
- Article 结构化数据；
- FAQ 内容；
- 统一作者简介；
- 律所信息；
- 文章页。
- robots.txt；
- sitemap.xml；
- llms.txt。
- 联系电话；
- 微信二维码。
- 公众号代表文章链接；
- 小红书笔记链接。

上线后建议提交到搜索引擎站长工具，并确保页面可公开访问，不需要登录。
