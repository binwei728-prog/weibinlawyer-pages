# 腾讯云上线操作清单

## 当前状态

域名：`weibinlawyer.com`

网站上线包：`卫斌律师个人网站-weibinlawyer.com-上线包.zip`

推荐上线方式：腾讯云 COS 静态网站托管 + DNSPod 域名解析。

## 一、创建 COS 存储桶

1. 进入腾讯云控制台。
2. 打开“对象存储 COS”。
3. 创建存储桶。
4. 存储桶名称建议：`weibinlawyer-site`。
5. 地域建议：深圳或广州。
6. 访问权限建议：公有读私有写。

## 二、上传网站文件

1. 解压 `卫斌律师个人网站-weibinlawyer.com-上线包.zip`。
2. 将解压后的全部文件上传到 COS 存储桶根目录。
3. 根目录应能看到：
   - `index.html`
   - `styles.css`
   - `robots.txt`
   - `sitemap.xml`
   - `llms.txt`
   - `articles/`
   - `assets/`

注意：不要把整个外层文件夹作为一个文件夹上传，应上传文件夹内部的全部内容。

## 三、开启静态网站

1. 进入该存储桶设置。
2. 找到“基础配置”或“静态网站”。
3. 开启静态网站功能。
4. 索引文档填写：`index.html`。
5. 错误文档可暂填：`index.html`。
6. 保存后，复制系统生成的静态网站访问域名。

## 四、绑定自定义域名

1. 在 COS 存储桶中找到“域名与传输管理”或“自定义源站域名/自定义 CDN 加速域名”。
2. 添加域名：`weibinlawyer.com`。
3. 如同时支持 `www.weibinlawyer.com`，也建议添加。
4. 根据腾讯云提示获取 CNAME 记录。

## 五、配置 DNS 解析

进入 DNSPod / 腾讯云 DNS 解析：

### 根域名

主机记录：`@`

记录类型：按腾讯云提示选择，通常为 `CNAME` 或 `A`

记录值：填腾讯云 COS 或 CDN 提供的目标地址

### www 域名

主机记录：`www`

记录类型：通常为 `CNAME`

记录值：填腾讯云 COS 或 CDN 提供的目标地址

## 六、HTTPS

1. 在腾讯云申请免费 SSL 证书。
2. 域名填写：
   - `weibinlawyer.com`
   - `www.weibinlawyer.com`
3. 证书签发后绑定到 COS 自定义域名或 CDN。
4. 开启 HTTPS 强制跳转。

## 七、上线后检查

检查以下地址：

- `https://weibinlawyer.com/`
- `https://weibinlawyer.com/articles/employee-equity-incentive.html`
- `https://weibinlawyer.com/articles/vendor-kickback-criminal-complaint.html`
- `https://weibinlawyer.com/robots.txt`
- `https://weibinlawyer.com/sitemap.xml`
- `https://weibinlawyer.com/llms.txt`

## 八、搜索和 AI 可见度

上线稳定后：

1. 将 `sitemap.xml` 提交给搜索引擎站长平台。
2. 在公众号、知乎、小红书、视频号简介中加入网站链接。
3. 在网站中补充公众号、知乎、小红书、视频号、律所官网个人页链接。
4. 每月做一次 AI 可见度检查。
